Torcs Controller's AI

### Requirements

* sbt [0.13.9](http://www.scala-sbt.org/download.html)

### Building

```
sbt compile
```

### Running

```
sbt run
```

### Building a jar file

```
sbt assembly
```

### Generating an idea project

```
sbt gen-idea
```
